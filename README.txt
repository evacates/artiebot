Hi Grayson, or even other future users. 
This is just a text document telling you what you probably need to know to edit Artie.
Just for some really quick coding basics.

An int (integer) is almost any whole number (1, 2, -52, 10000)
A double is any number with a decimal place (1.0, 2.46, -3.14)
A char (character) is a single character ('a', 'b', 'C')
A String is a string of characters ("Hello", "Bye", "I'm going to the mall, want to come?")
Note: Strings are wrapped by "double quotations" chars are wrapped by 'single quotations'

config.py
Anytime you want to add more roles or channels to Arties domain, you will need to declare them in config.py.
This means if you want to hook up another role react button, or set up specific bot channel things, you will need to go there first to define
what channel you are using or what roles he needs to know.
